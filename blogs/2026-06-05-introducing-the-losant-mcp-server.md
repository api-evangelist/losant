Losant has released a new MCP Server that enables analysis of application configurations and telemetry data within preferred AI tools, expanding its AI-powered features for IoT developers. The Model Context Protocol Server allows users to connect their IoT data with major large language models like Claude, ChatGPT, and Google Gemini. The hosted service is available via OAuth 2.0 authentication for querying and gaining insights into Losant IoT applications.
